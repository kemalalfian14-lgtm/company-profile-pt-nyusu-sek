# Kontribusi Saya

## Kemal Alfian Falhaqi

Pada pengembangan website **Company Profile PT. NYUSU SEK**, saya berperan sebagai **Backend Developer** yang bertanggung jawab dalam membangun sistem aplikasi menggunakan framework **Laravel 12**. Fokus utama saya adalah mengembangkan logika bisnis aplikasi, mengelola database, membangun fitur administrasi, serta memastikan seluruh proses pengolahan data dapat berjalan secara optimal, aman, dan efisien.

Selama proses pengembangan, saya tidak hanya mengimplementasikan fitur-fitur utama aplikasi, tetapi juga merancang struktur sistem agar mudah dikembangkan, dipelihara, dan digunakan oleh pengguna. Berikut merupakan rincian kontribusi yang saya lakukan.

---

## 1. Backend Development

Saya bertanggung jawab dalam mengembangkan keseluruhan proses backend aplikasi menggunakan framework Laravel. Seluruh logika bisnis, proses pengolahan data, validasi, hingga komunikasi antara tampilan (View) dengan database dibangun menggunakan arsitektur Laravel yang menerapkan konsep MVC (Model-View-Controller).

Selain itu, saya memastikan setiap fitur dapat berjalan sesuai kebutuhan pengguna dengan memperhatikan efisiensi kode, kemudahan pemeliharaan (maintainability), serta keamanan aplikasi.

Kontribusi pada bagian ini meliputi:

- Membangun struktur aplikasi Laravel.
- Mengembangkan logika bisnis pada setiap modul.
- Menghubungkan tampilan dengan database.
- Mengoptimalkan proses pengolahan data.
- Menjaga konsistensi struktur kode sesuai standar Laravel.

---

## 2. Authentication System

Saya mengembangkan sistem autentikasi yang berfungsi untuk membatasi hak akses pengguna terhadap halaman administrator. Sistem ini memastikan hanya pengguna yang memiliki akun dan telah berhasil melakukan login yang dapat mengakses dashboard administrasi.

Implementasi autentikasi meliputi:

- Halaman Login Administrator.
- Validasi username dan password.
- Manajemen Session.
- Logout System.
- Middleware untuk membatasi akses halaman admin.
- Pengamanan route administrator.

Dengan adanya fitur ini, keamanan aplikasi menjadi lebih terjamin karena data hanya dapat dikelola oleh pengguna yang memiliki hak akses.

---

## 3. CRUD (Create, Read, Update, Delete)

Saya mengembangkan seluruh fitur CRUD yang digunakan administrator untuk mengelola data pada website.

Fitur CRUD memungkinkan administrator untuk:

- Menambahkan data baru.
- Menampilkan seluruh data.
- Melihat detail data.
- Mengubah informasi yang sudah ada.
- Menghapus data yang tidak diperlukan.

Setiap proses CRUD dilengkapi dengan validasi data agar informasi yang tersimpan di dalam database tetap konsisten dan meminimalkan terjadinya kesalahan input.

---

## 4. Database Design & Management

Saya merancang struktur database yang digunakan oleh aplikasi, mulai dari menentukan tabel, atribut, relasi antar tabel, hingga tipe data yang digunakan.

Dalam proses ini saya juga memperhatikan normalisasi database agar data tidak mengalami redundansi dan proses pengambilan data menjadi lebih efisien.

Kontribusi pada bagian database meliputi:

- Perancangan Entity Relationship.
- Penentuan struktur tabel.
- Relasi antar tabel.
- Optimasi penyimpanan data.
- Pengelolaan data menggunakan Eloquent ORM.

---

## 5. Controller Development

Saya mengembangkan Controller sebagai pusat logika aplikasi yang menghubungkan tampilan dengan database.

Controller yang saya bangun bertugas untuk:

- Menerima request dari pengguna.
- Melakukan validasi data.
- Memproses logika bisnis.
- Mengirim data ke View.
- Menyimpan perubahan ke database.
- Menangani proses CRUD secara keseluruhan.

Dengan pembagian tugas tersebut, struktur aplikasi menjadi lebih rapi dan mengikuti pola pengembangan Laravel.

---

## 6. Model Development

Saya mengembangkan Model menggunakan Eloquent ORM Laravel sebagai media komunikasi antara aplikasi dengan database.

Melalui Model, seluruh proses manipulasi data dilakukan secara lebih sederhana tanpa harus menuliskan query SQL secara manual.

Kontribusi pada bagian Model meliputi:

- Membuat Model setiap tabel.
- Menentukan relasi antar Model.
- Mengelola proses pengambilan data.
- Mengelola penyimpanan data.
- Mengelola pembaruan dan penghapusan data.

---

## 7. Database Migration

Saya membuat serta mengelola Migration Laravel untuk membangun struktur database secara otomatis.

Migration memudahkan proses instalasi aplikasi karena struktur database dapat dibuat kembali hanya dengan menjalankan perintah migration tanpa harus membuat tabel secara manual.

Kontribusi meliputi:

- Membuat file migration.
- Menentukan struktur tabel.
- Menentukan tipe data setiap kolom.
- Menentukan primary key dan foreign key.
- Mengatur relasi antar tabel.
- Memastikan database dapat dibangun kembali secara konsisten pada berbagai lingkungan pengembangan.

---

## Hasil Kontribusi

Melalui kontribusi tersebut, saya berhasil membangun sistem backend website **Company Profile PT. NYUSU SEK** yang mampu mendukung seluruh proses pengelolaan data perusahaan secara efektif. Sistem yang dikembangkan telah menerapkan konsep MVC Laravel, memanfaatkan Eloquent ORM untuk pengelolaan database, menyediakan fitur autentikasi yang aman, serta mendukung proses CRUD secara lengkap sehingga memudahkan administrator dalam mengelola konten website.
